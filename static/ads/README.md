# Advertisement Files Directory

## Directory Structure

This directory contains advertisement files organized by target gender:

- `male/` - Ads targeted at male audience
- `female/` - Ads targeted at female audience  
- `neutral/` - Ads for general audience

## Supported Formats

- **Images**: `.jpg`, `.jpeg`, `.png`, `.gif`
- **Videos**: `.mp4`, `.webm`

## How to Add Ads

1. Place image or video files in the appropriate gender directory
2. Files are automatically loaded on server startup
3. Ads are randomly selected based on detected gender

## Example Files

For testing, you can add:
- Sample male product images/videos to `male/`
- Sample female product images/videos to `female/`
- General promotions to `neutral/`

## File Naming

Files can have any name, but recommended format:
- `product_name_type.ext` (e.g., `men_fashion_watch.jpg`)
- `brand_campaign_gender.ext` (e.g., `nike_sports_male.mp4`)
